Say hello to akka-http

Defined resources:

  • / - Redirect using HTTP 302 - Found - to a resource in sbt-web project
  • /web/webui/0.0.1/index.html - serving static contents from WebJars in resources. See /web/react/0.12.2/main.js for example - dependency on 
    "org.webjars" % "react" % "0.12.2"
  • /json - Serialization using spray-json
  • /ping - Return result directly from the route
  • /coreping - ping from actor located in another project
  • /secure Use any username and '<username>-password' as credentials
  • /crash
  • /shutdown - never do this in production :-)
